* MS SqlServer 2008 normalmente este viene incluido con Visual Studio 2010

Pasos:

1) En la carperta imagenes, podemos encontrar las imagenes de referencias para crear la base de datos mucho mas rapida.

2) Primero en la pestaña datos de Ms Visual Studio 2010,  creamos una nueva consulta de base de datos como se  muestra
en la imagen 1.

3) Debemos seleccionar el string de conexion a base de datos pertinente a nuesto equipo en mi caso joe, que es mi nombre
de Pc en windows.

4) Al presionar el boton conectar, se abrira una ventana donde podemos realizar nuestro query de base de datos, en este
debemos ingresar la consulta que esta en el script.sql que esta en la carpeta sql de nuestro proyecto.

5) al ejecutarlo en el boton de lanzar consulta como se muestra en la imagen 4.

6) en este punto la base de  datos debe estar creada, y podemos comprobarlo como nos muestra en la imagen 5, creando una 
revisando la conexion a la base de datos desde explorador de servidores en la pesataña de conexion que hemos creado
buscando las tablas y que existan en la base de datos llamada twisted.
